Subnetting Without Tears: How CIDR Finally Clicked

Why it kept not clicking

I was trying to do binary in my head under time pressure. That works for exactly nobody.

The fix was to stop thinking in bits and start thinking in block sizes — the number of addresses each subnet owns.

The 4-step method

  1. Find the block size: 256 - the interesting octet.
  2. List the subnets in jumps of the block size.
  3. The broadcast is one below the next subnet.
  4. Usable hosts are everything in between.
192.168.1.0/26   -> hosts .1-.62,   broadcast .63
192.168.1.64/26  -> hosts .65-.126, broadcast .127

Tip: if your network number lands on an odd value for /25+, you've slipped a block.
